En primer lugar, lo que ms se puede destacar es que el verano representa el periodo de menor consumo energtico, destacando especialmente el mes de agosto, cuando todos los sectores muestran una cada pronunciada. Este comportamiento es coherente con las vacaciones estivales y los cierres temporales, especialmente en los sectores secundario y terciario. Seguidamente, el sector terciario es el mayor consumidor de energa a lo largo de todo el periodo analizado. Su consumo se mantiene relativamente estable, con valores que oscilan entre los 4.200 y 4.500 puntos excepto en los periodos vacionales. En cuanto al sector secundario es el segundo mayor consumidor de energa por detrs del sector terciario y su consumo es el que ms varia de los tres. Esto se debe a que las actividades industriales suelen ajustarse a la demanda del mercado, lo que provoca aumentos o reducciones estacionales en su produccin. Por ltimo, vemos un sector primario con poca influencia en el consumo elctrico en Espaa. Al compararlo con el consumo del 2015, no hay grandes diferencias remarcables a primera vista. No obstante, se aprecia una ligera disminucin del consumo energtico en la industria, lo cual podra deberse a una desindustrializacin parcial en algunos sectores especficos

# Interfaz de usuario
ui <- fluidPage(
  titlePanel("Mapa interactivo de valores por pas en Europa"),
  sidebarLayout(
    sidebarPanel(
      selectInput("anio", "Selecciona el ao:", 
                  choices = sort(unique(datos_europa$TIME_PERIOD)),

    ),
    mainPanel(
      plotOutput("mapa", height = "600px", width = "100%")
    )
  )
)

# Lgica del servidor (genera dinmicamente el mapa en base al ao seleccionado)
server <- function(input, output) {
  output$mapa <- renderPlot({
    # Filtrar datos por ao seleccionado
    datos_filtrados <- datos_europa %>% filter(TIME_PERIOD == input$anio)
    
    # Combinar con mapa
    mapa_europa <- europe %>%
      left_join(datos_filtrados, by = c("name" = "geo"))
    
    # Graficar
    ggplot(data = mapa_europa) +
      geom_sf(aes(fill = OBS_VALUE)) +
      scale_fill_gradient(name = "Valor observado",
                          low = "lightyellow", high = "darkred", na.value = "grey90") +
      labs(title = paste("Mapa de calor - Ao:", input$anio),
           subtitle = "Coloreado segn OBS_VALUE",
           caption = "Unin Europea") +
      coord_sf(xlim = c(-25, 45), ylim = c(34, 72), expand = FALSE) +
      theme_minimal()
  })
}

# Ejecutar la aplicacin
shinyApp(ui = ui, server = server)
